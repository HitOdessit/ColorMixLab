# Support

## Where to get help

- **Bugs and unexpected behavior** — Open a [Bug Report](https://github.com/HitOdessit/ColorMixLab/issues/new?template=bug_report.md).
- **Feature requests** — Open a [Feature Request](https://github.com/HitOdessit/ColorMixLab/issues/new?template=feature_request.md).
- **General questions** — Open a regular issue with a clear title; please check existing issues first.
- **Architecture / design discussion** — Read [ARCHITECTURE.md](ARCHITECTURE.md), then open an issue with the `discussion` label.
- **AI build process questions** — Read [CLAUDE.md](CLAUDE.md) and [docs/reproduce.md](docs/reproduce.md), then open an issue.
- **Security vulnerabilities** — See [SECURITY.md](SECURITY.md). Do not open a public issue.

## Response expectations

This is a personal portfolio project. The maintainer aims to respond to issues within 7 days but cannot guarantee a fix timeline. PRs with tests are the fastest path to landing a change.
