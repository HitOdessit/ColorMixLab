# Development notes (curated AI-generated archive)

These are seven representative markdown files generated by Claude Code during the development of ColorMixLab. They were curated from a larger set of 60+ files the AI produced as it wrote the code — a real-time engineering journal of features, fixes, and migrations.

The full original set (animation tweaks, layout iterations, throwaway build instructions, duplicate "implementation complete" summaries) was archived out. What's kept here is the substantive subset: each file documents a meaningful step in the build.

## Why keep these at all

Two reasons:

1. **Honesty about the AI-build process.** A polished portfolio repo could quietly delete every artifact. Keeping a curated set visible shows the unfiltered shape of AI-assisted development — including the places where features needed multiple iterations.
2. **Useful artifacts.** Each kept file documents a non-trivial decision point or design — useful for someone reading the codebase or attempting similar work.

## What's missing

These are *outputs*, not *inputs*. They document what the AI built; they don't expose the prompts that produced them. For example prompts you could use to reproduce parts of this project, see [`docs/reproduce.md`](../reproduce.md).
